### macOS configuration

- [Iterm2](https://iterm2.com/) : Terminal
- [omz](https://ohmyz.sh/) : Shell
- [AstroNvim](https://docs.astronvim.com/) : Editor
- [RayCast](https://www.raycast.com/): App launcher
- [AutoRaise](https://github.com/sbmpost/AutoRaise) : Focus follow cursor
- [Karabiner](https://karabiner-elements.pqrs.org/): Binding fn as meta key & Caps as escape
- [Aerospace](https://github.com/nikitabobko/AeroSpace): Window manager
- [JankyBorder](https://github.com/FelixKratz/JankyBorders): Add border to windows
